# BRIX Money — Fonts

This package does not include font files due to licensing restrictions. Below are the recommended sources for obtaining the required fonts.

## Typography Stack

### 1. Founders Grotesk (Display/Titles)
**License**: Custom — Contact BRIX Money team for access  
**Usage**: Campaign headlines, brand statements, premium moments

**Obtain from**:  
- Internal brand assets (contact brand team)
- [Founders Grotesk](https://www.foundersgrotesk.com/)

### 2. Urbanist (Body/UI)
**License**: Open Font License (OFL)  
**Usage**: Body copy, system UI, functional text

**Install via Google Fonts**:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Urbanist:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

**CSS**:
```css
font-family: 'Urbanist', sans-serif;
```

### 3. Nyght Serif (Editorial/Accent)
**License**: Custom — Contact BRIX Money team for access  
**Usage**: Selective premium moments, editorial content

**Obtain from**:  
- Internal brand assets (contact brand team)

---

## Usage Guidelines

- **Founders Grotesk**: Use for campaign-level statements only
- **Urbanist**: Default for all functional copy and UI elements
- **Nyght Serif**: Use sparingly for premium/editorial emphasis

Refer to `guidelines.md` for complete typography hierarchy and usage rules.
