## 2) Logo System

The guideline defines three core logo components:
- Symbol
- Logotype
- Wordmark

It shows:
- Standalone symbol treatment
- Wordmark/logotype combinations
- Mixed mark compositions
- Light and dark background applications

Pattern usage is also defined:

## 3) Color System

### Primary palette and meaning
- **Brix Orange** `#DF7032` (`223, 112, 50`)  
  Meaning: emerging-market energy and growth dynamics
- **Nightly / Deep Black-Slate** `#0B0B0B` (`11, 11, 11`)  
  Meaning: institutional stability and complexity simplified
- **White / Off-White** `#FFFFFF` (`255, 255, 255`)  
  Meaning: transparency of onchain settlement and real-time NAV verification
- **Platinum** `#E6E6E6` (`230, 230, 230`)  
  Meaning: neutral support/background tone

### Brand gradient
- **Branding Gradient** from `#FFFFFF` to `#DF7032` (linear)
- Additional dark/orange gradient usage is present for branded backgrounds and emphasis blocks

### Color usage hierarchy (as represented in pages)
- Orange as high-impact accent and branded containers
- Nightly as authority/contrast base
- White for clean foreground and transparency cues
- Platinum for soft neutral surfaces
- Gradient for hero/emphasis/brand-expression zones

### Color usage examples included
- Slogan contrast checks (`Access outlasts control.`) across dark/light/neutral surfaces
- Logo visibility on dark and light fields
- Card-like panels in black, platinum, and white

## 4) Typography System

Three type families are defined and repeatedly tested:

### A) Founders Grotesk (Custom)
- Role: Main title/display font
- Seen in: large headlines, wordmarks, typographic hero compositions
- Specimen includes:
  - Uppercase/lowercase/symbols
  - Large hero characters (`b R x`)
  - Full numeric set (`0-9`)
  - Full alphabet set (`A-Z`)

### B) Urbanist (Google Font)
- Role: Main body/system supporting font
- Typical body style in guide copy: `16px`, medium/500 weight
- Also used in high-impact headline experiments and specimen testing
- Specimen includes:
  - Full numeric set (`0-9`)
  - Full alphabet set (`A-Z`)
  - Large `b R x` test forms

### C) Nyght Serif (Custom)
- Role: Editorial/expressive serif for premium narrative moments
- Used for:
  - Select title treatments
  - Alternate body/display specimen page
  - Full character demonstrations (digits, alphabet, symbols)

### Typography tone rules embedded in the guideline
- Headlines: `Bold, authoritative, and economically literate.`
- Body: `Precise and disciplined. Avoid buzzword padding; focus on rate differentials and monetary policy.`

## 9) Implementation Notes for Brand Consistency

- Keep core palette strict: `#DF7032`, `#0B0B0B`, `#FFFFFF`, `#E6E6E6`.
- Use Founders Grotesk for primary titles and campaign-level statements.
- Use Urbanist for functional copy, UI, and body readability.
- Use Nyght Serif selectively for premium/editorial emphasis.
- Preserve high-contrast compositions and avoid low-contrast color pairings.
- Use symbol repetition as a controlled motif, not random decoration.
- Retain concise, financially literate voice in all brand communications.

- **Nyght Serif** (`.otf`)
- **Urbanist** (`.ttf`, including variable fonts)

### Logos attached
- **PNG set** for Light and Dark variants:
  - Full logo
  - Logotype
  - Symbol
  - Background-adapted exports (`Logo Light BG`, `Logo Dark BG`, `Symbol Light BG`, `Symbol Dark BG`)
- **SVG set** for Light and Dark variants:
  - Full logo
  - Logotype
  - Symbol
  - Alternate `Logo` and `Symbol` files

### Practical usage rule
- Use `SVG` files for product/UI/web implementation whenever possible.
- Use `PNG` files for quick docs, slides, and environments without SVG support.
- Match logo variant to background contrast (`Light` assets on dark surfaces, `Dark` assets on light surfaces).
